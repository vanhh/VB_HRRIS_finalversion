Algorithm

Inserting a record in the booking table

Once user finishes selecting room type, checkin and chekout date, the application will search database for available rooms that suit user's needs. The room list is shown as a dropdown so that user can choose easily. When a particular room is chosen, based on the room number,the application will automatically get the price as well as calculate total price as display it on the screen (function in Booking Controller). If there is no room avalabile, a message box will show up to let user know.
When user click Insert, the application will start checking all the fields firstly (Validation class). For instance, room number and customer id cannot be empty. Number of guests must be numeric value and in the allowed range for each room type. If all fields are valid, the applicatin will insert to database with the given input. Otherwise,error tooltips with message will be shown to alert user about their error and let them correct.

Instructions on booking a room

  1. Select Room Type from dropdown list
  2. Select Checkin - Checkout date. By default, they are the current date
  3. Click Search to find the available rooms during that period
  4. Select Room Number from dropdown list. Price, Number of days, and Total are automatically generated.
  5. Enter customer's first name and last name
  6. Click Search to get the customer ID. If the customer profile is not on the database, click Yes button on pop up message to create a new profile
  7. Select Booking Date which is set to current date by default
  8. Enter Number of guests
  9. Enter comments if customer prefer
  10. Click Insert. Correct input field if there are any invalid ones
