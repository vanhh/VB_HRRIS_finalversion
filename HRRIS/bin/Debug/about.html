About Hotel Room Reservation Information System

The application is used to help not only hotel owner manage their business but also the staff increase work effieciency. The application allows staff or aminstration to manage customer, room and booking information. They can also generate report to see the status and summary of their business.

Features for Customer/Room Managment

  • Create new Customer/Room profile
  • View and Update existing room profile
  • Search for customer /room profile
  • Generate report on Booking history of a specific customer/room

Features for Booking Managment

  • Create new booking reservation
  • View and Update existing booking
  • Search for booking
  • Generate control break report of booking during a given year month
  • Generate report on Booking History of a given month year
